## Step 2. 로그인

먼저 `relay whoami`로 로그인 상태를 확인합니다. 이미 로그인되어 있으면 이 단계를 건너뛰세요.

로그인이 필요하면:

```bash
relay login
```

- 브라우저가 자동으로 열리면 GitHub 또는 카카오 계정으로 로그인합니다.
- 브라우저가 열리지 않으면 CLI 출력에서 로그인 URL을 찾아 플랫폼에 맞는 명령어로 직접 엽니다:
  - macOS: `open <URL>`
  - Linux: `xdg-open <URL>`
  - Windows: `start <URL>`

브라우저를 연 뒤, 사용자 질문 도구를 호출하여 로그인 완료를 확인하세요:
- question: "브라우저에서 로그인을 완료하셨나요?"
- options: ["완료했습니다", "도움이 필요합니다"]

"완료했습니다" → `relay whoami`로 로그인 성공을 확인한 후 다음 단계로 진행
"도움이 필요합니다" → 문제 상황을 파악하고 안내
