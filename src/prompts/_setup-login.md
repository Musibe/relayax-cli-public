## Step 2. 로그인

먼저 `relay whoami`로 로그인 상태를 확인합니다. 이미 로그인되어 있으면 이 단계를 건너뛰세요.

로그인이 필요하면:

**중요: `relay login`은 로컬 콜백 서버를 띄우고 브라우저 로그인 완료를 기다리는 명령어입니다.**
- 반드시 포그라운드에서 실행하고, timeout을 300초(5분) 이상으로 설정하세요.
- 명령어가 종료되기 전에 사용자가 브라우저에서 로그인을 완료해야 합니다.
- 명령어를 먼저 종료하면 콜백 서버가 사라져 로그인이 실패합니다.

```bash
relay login
```

- 브라우저가 자동으로 열리면 GitHub 또는 카카오 계정으로 로그인합니다.
- 브라우저가 열리지 않으면 CLI 출력에서 로그인 URL을 찾아 플랫폼에 맞는 명령어로 별도 실행하여 엽니다:
  - macOS: `open <URL>`
  - Linux: `xdg-open <URL>`
  - Windows: `start <URL>`

`relay login`이 "✓ 로그인 완료"를 출력하면 다음 단계로 진행합니다.
실패하면 `relay whoami`로 상태를 확인하고, 필요시 재시도합니다.
